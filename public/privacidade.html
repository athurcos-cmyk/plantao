Política de Privacidade

Última atualização: março de 2026

1. Sobre o Plantão

O Plantão é um aplicativo web progressivo (PWA) criado para auxiliar profissionais de enfermagem a registrar anotações durante o plantão hospitalar. Desenvolvido e operado por Arthur Thurcos, com sede no Brasil.

Contato: contato@plantao.net

2. Dados coletados

Coletamos apenas o necessário para o funcionamento do app:

  • Nome e e-mail — para criar e identificar sua conta
  • Anotações clínicas — texto que você digita no app (sinais vitais, medicações, etc.)
  • Dados de pacientes — nome e leito, informados por você
  • Token de notificação (FCM) — para enviar lembretes de pendências no celular
  • Dados de uso básico — data de cadastro, último acesso

Não coletamos prontuários, CPF, dados financeiros ou qualquer informação sensível além do que você digita voluntariamente.

3. Como os dados são usados

  • Sincronizar suas anotações entre dispositivos
  • Enviar notificações de pendências que você agendou
  • Enviar e-mails transacionais: boas-vindas, dicas e despedida
  • Melhorar o funcionamento do app com base em feedbacks

Não vendemos, compartilhamos nem monetizamos seus dados com terceiros.

4. Armazenamento

Seus dados são armazenados no Firebase Realtime Database (Google), com servidores nos Estados Unidos. O acesso é protegido por autenticação Firebase Auth e regras de segurança que isolam os dados de cada usuário.

5. Dados de pacientes

O Plantão é uma ferramenta de apoio ao profissional — não é um sistema hospitalar. Os dados de pacientes (nome, leito, anotações) são inseridos por você e ficam restritos à sua conta. Não temos acesso ao conteúdo das suas anotações. Você é responsável por seguir as normas do seu hospital quanto ao registro de informações clínicas.

6. Seus direitos

  • Acesso: todos os seus dados estão visíveis dentro do app
  • Exclusão: em Configurações → Excluir conta, todos os seus dados são removidos permanentemente
  • Portabilidade: entre em contato em contato@plantao.net para exportar seus dados

7. Cookies e armazenamento local

O app usa localStorage para cache de sessão e funcionamento offline. Não usamos cookies de rastreamento ou publicidade.

8. Menores de idade

O Plantão é destinado a profissionais de saúde maiores de 18 anos. Não coletamos dados de menores de idade.

9. Alterações

Atualizações nesta política serão comunicadas por e-mail. O uso continuado do app após a notificação implica aceite das mudanças.
